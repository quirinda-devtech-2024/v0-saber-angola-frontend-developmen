# Estratégia de Aquisição dos Primeiros 1000 Usuários - SaberAngola

## 1. Fases de Aquisição

### 1.1. Fase 1: 0-100 Usuários (Soft Launch)
```yaml
Objetivo:
  - Validar produto
  - Coletar feedback inicial
  - Identificar bugs
  - Ajustar UX/UI
  - Tempo estimado: 2 semanas

Estratégias:
  1. Network Pessoal:
    - Amigos universitários
    - Colegas de trabalho
    - Familiares em idade escolar/universitária
    - Meta: 20 usuários

  2. Grupos Universitários:
    - Grupos WhatsApp da universidade
    - Associações estudantis
    - Centros acadêmicos
    - Meta: 40 usuários

  3. Cyber Cafés:
    - Parcerias com 5 cyber cafés
    - Demonstrações gratuitas
    - Material promocional
    - Meta: 40 usuários

Custo Estimado: 0 KZ
```
